# Part 1 – Course Information

Solutions for exercises 1.1–1.2 of [Full Stack Open](https://fullstackopen.com/en/part1/introduction_to_react#exercises-1-1-1-2).

## Exercises

### 1.1 – Course Information, step 1
Initialized a React app with Vite and refactored a single-component app into three components: `Header`, `Content`, and `Total`. Data is defined in `App` and passed down via props.

### 1.2 – Course Information, step 2
Refactored `Content` to render three `Part` components instead of handling the output directly. Each `Part` receives the name and exercise count of one course part as props.
